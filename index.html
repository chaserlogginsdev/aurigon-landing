The problem

Your accounts are a mess. You just can't see it yet.

Most small IT teams manage user accounts by RDPing into each machine one at a time. There's no central view, no audit trail, and no easy way to know what you're missing until something goes wrong.

👻
Ghost accounts pile up

Ex-employees, temporary vendors, test users — they accumulate with nobody watching. Each one is an open door.

🔍
No visibility across machines

You have no idea how many accounts exist across your fleet, which are active, or which shouldn't be there.

📋
No audit trail

Who disabled that account? When? Without a log, you're reconstructing history from memory — if at all.

⏱️
Manual work that doesn't scale

RDP and click works for 10 machines. For 50 it's a full-time job — and it leaves no record behind.

Features

Everything you need. Nothing extra.

Central account dashboard

See every local account across every machine in one place. Filter by status, role, or search by username and SID.

Remote account management

Create, disable, enable, or delete accounts from the dashboard. Actions execute within 30 seconds — no RDP needed.

Group management

View all local groups and their members. Add or remove users, create new groups — all from one interface.

Full audit log

Every action is recorded — who triggered it, when, on which machine, and what the result was. Always searchable.

Role-based access

Admins can make changes. Viewers can see everything but can't touch anything. Right access for every team member.

Lightweight agent

A small binary runs on each machine. It reports inventory every 30 seconds and executes queued actions. No bloat.

How it works

Up and running in minutes.

No Active Directory required. No domain membership. Works on any Windows machine in your environment.

01
Deploy the agent

Drop the Aurigon agent on each machine. Set two environment variables — your server URL and an agent key — and run it.

02
Machines appear automatically

Each agent registers itself and begins uploading account and group inventory every 30 seconds. Your machines show up in the dashboard right away.

03
Manage from one place

Create accounts, disable old ones, manage groups — all from the dashboard, without touching individual machines.

04
Every action is logged

The audit log captures who did what, when, and what the result was. When someone asks "who did that?" — you'll know.
